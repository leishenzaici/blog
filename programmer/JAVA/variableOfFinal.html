一、常量和变量

1、常量和变量的区别

（1）常量的值不能发生变化。

（2）变量的值是可以发生变化的。

2、规范使用

（1）声明常量需要使用final 关键字，命名需要全部用大写字母，单词之间用下划线分隔，例如：


        final int BAIDU_URL="www.baidu.com";

（2）变量声明使用驼峰命名法，例如：


        int studentAge=28;
